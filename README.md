## 📊 Sobre el Dataset

Base de datos sobre el bienestar psicológico en investigadores académicos y estudiantes universitarios (pregrado y posgrado) a nivel mundial y por regiones para el período 2019-2024.

- **Observaciones:** 64 | **Variables:** 13 | **Regiones:** 6
- **Fuente:** [Harvard Dataverse — DOI: 10.7910/DVN/EZP0YO](https://doi.org/10.7910/DVN/EZP0YO)
- **Autor:** de la Serna, Juan Moises (UNIR) | [ORCID: 0000-0002-8401-8018](https://orcid.org/0000-0002-8401-8018)
- **Licencia:** CC0 1.0

**Fuentes:** Nature PhD Survey (n=6.063), Wellcome Trust (n>4.000), OECD 2023, PMC/PubMed, ASEAN J. Psychiatry.
